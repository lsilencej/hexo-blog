微软自家的远程桌面？ RD Client

前言

看到一篇有关远程协作的文章，突然想起微软自家有个 RD Client ，便选择下载下来玩玩，顺便看看效果，安装过程还是挺简单的

准备

电脑

在设置中找到远程桌面选项并打开

手机

下载 RD Client 软件，Google 商店有（新版，听说卡带宽限制，可下载 8 开头的版本）

配置

内网

  • 点击右上角的 ‘ + ‘ 号，Add PC

    点击 ADD MANUALLY

  • PC NAME 填写电脑的IP地址，可以通过命令行 cmd ，输入 ipconfig 查询，一般为 IPv4 的地址，形如 192.168.1.1

  • 下面的 USER ACCOUNT 为你的 Microsoft 账户，登录电脑的时候需要用到，默认为连接时询问，可以直接添加账户

  • 右上角 SAVE 保存即可

  • 点进去连接即可

    然后就可以在手机上远程操控 PC 了

frp内网穿透

frp内网穿透和内网的步骤没差多少，只是需要把 PC NAME 填写的电脑 IP 改成内网穿透后的 IP

我用的是 sakura frp，每天签到白嫖流量（笑，访问密钥用来登录桌面客户端

sakura frp

  • 下载桌面客户端

  • 创建新隧道

  • 打开隧道，日志中有一个使用 ip 地址连接，把ip地址填入之前的 PC NAME 当中即可

总结

RD Client 相较于 Steamlink 来说对电脑的适配性比较好点，总的来说还是比较流畅的，不过无法连接手柄，有连接手柄的需求还是选择 Steamlink ，晚上躲在被窝里偷偷推 Gal 不香吗，远程桌面简直了
